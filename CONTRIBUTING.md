# Contributing

Thanks for contributing to `berlinwahl-abgeordnetenhaus`.

This repository is a curated list of official datasets, maps, timelines, legal references, and related resources for the Berlin elections on 20 September 2026.

## What To Add

Add resources that are directly relevant to the 2026 Abgeordnetenhaus and BVV elections in Berlin, for example:

- official open datasets
- geometry files, maps, and boundary references
- timelines, forms, and legal guidance
- voter information pages and FAQs
- official operational material from the Landeswahlleiter or Berlin administration

## Entry Guidelines

Each entry should:

- link directly to the dataset, document, or official page
- stay focused on Berlin's 2026 election cycle
- have a short description explaining why the resource matters
- include important dates or scope where useful
- stay concise and factual

## Before Opening A Pull Request

Please check that:

- the resource is clearly about the Berlin 2026 election
- it is added to the most appropriate section
- it is not already listed elsewhere in the file
- the description matches the style of nearby entries
- the link works

## Updating Existing Entries

Improvements are welcome, especially:

- fixing broken links
- clarifying descriptions
- adding newly published official datasets
- marking outdated pre-election guidance
- adding better primary-source links

## Pull Requests

Small, focused pull requests are preferred. If a resource is only loosely connected to the 2026 election, leave it out or open an issue first.
